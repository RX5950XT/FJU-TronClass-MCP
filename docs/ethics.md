# 學業誠信聲明與使用規範

## 工具用途

本工具（`fju-tronclass-mcp`）為個人學習輔助工具，旨在提升學習效率：

- 快速查閱課程資訊、公告、待辦事項
- 下載授課教材以供離線閱讀
- 透過 AI assistant（Claude 等）管理學習進度

---

## 影片完成度功能（`fju_mark_video_complete`）

此功能可將 TronClass 學習活動的觀看進度標記為完成（completeness = 100）。

### 使用者自負責任聲明

**使用此功能前，請確認您已知悉以下事項：**

1. **繞過觀看驗證**：此功能直接呼叫後端 API 上報進度，不需實際播放影片
2. **學業誠信風險**：若課程規定「必須觀看影片才能取得出席或成績」，使用此功能可能違反課程要求
3. **校規責任**：是否違反輔仁大學學術誠信規範，由使用者自行判斷並承擔後果
4. **建議用途**：適合用於已觀看過內容、需重新標記完成度的情境，或對課程內容已充分理解的情況

**本工具作者不對任何學業處分或相關後果負責。**

---

## 教材下載

- 下載之教材**僅供個人學習使用**
- **請勿散佈**或再製授課教材，以尊重教師著作權
- 部分課程可能設定 `allow_download: false`，本工具可繞過此限制；使用者應自行判斷是否符合課程政策

---

## 技術使用限制

- 本工具基於逆向工程已驗證的 API endpoints，學校隨時可能修改 API
- 請勿用於大量自動化請求（可能導致帳號被封鎖）
- 請勿用於存取他人帳號資料
